Orion Mesh Overview
A self-healing mesh network for industrial sensor deployments. Connects up to 500 nodes across 2km² with sub-10ms latency and automatic failover.
Architecture
Each deployment consists of Nodes (OM-200/400), a Gateway Hub, and the Orion Cloud dashboard for fleet management and telemetry.
